# فاتورة بيع مجدولة

## الوصف
شاشة إنشاء وإدارة فواتير البيع المجدولة حسب الاتفاقيات.

## المميزات
- إنشاء فواتير بيع مجدولة
- ربط الفاتورة بالاتفاقية
- تحديد مواعيد الفواتير
- إدارة شروط الدفع
- تتبع حالة الفواتير

## الحقول المطلوبة
- بيانات العميل
- رقم الاتفاقية
- تفاصيل البضائع
- مواعيد الفواتير
- شروط الدفع

## الاستخدام
تستخدم لإنشاء فواتير بيع مجدولة حسب الاتفاقيات المبرمة مع العملاء.
